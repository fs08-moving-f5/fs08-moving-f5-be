## 🎯 서비스 소개

**무빙(Moving)**은 이사 전문가와 소비자를 효율적으로 연결하는 매칭 플랫폼입니다.

> 프로젝트 기간: 2025. 12. 05. - 2026. 01. 26.

### 주요 기능

- **견적 요청 및 비교**: 원하는 이사 조건을 입력하고 여러 기사님의 견적을 한눈에 비교
- **전문가 매칭**: 지역, 서비스 유형, 경력 등을 기반으로 최적의 이사 전문가 추천
- **후기 및 평점**: 실제 고객 후기와 평점을 통해 신뢰할 수 있는 전문가 선택
- **즐겨찾기**: 마음에 드는 기사님을 즐겨찾기에 추가하여 나중에 쉽게 찾기
- **실시간 알림**: 견적 요청, 수락, 거절 등 실시간 알림으로 빠른 소통
- **프로필 관리**: 사용자와 기사님 모두 자신의 프로필을 관리하고 업데이트

### 서비스 유형

- 일반 이사 (원룸, 투룸, 아파트 등)
- 기업/사무실 이사
- 포장 이사

### 주요 디렉토리 설명

- **`src/api/`**: 기능별 API 엔드포인트 (controller, service, repository, routes, validators 포함)
- **`src/config/`**: 데이터베이스, Redis, AWS 등 외부 서비스 설정
- **`src/middlewares/`**: 인증, 에러 처리, 검증 등 공통 미들웨어
- **`src/cache/`**: Redis 캐시 관리 및 무효화 로직
- **`prisma/`**: 데이터베이스 스키마 및 마이그레이션 관리

## 🚀 시작하기

### 사전 요구사항

- Node.js 24.x 이상
- PostgreSQL 데이터베이스
- Redis 서버
- AWS S3 (이미지 업로드용)

### 설치

```bash
# 의존성 설치
npm install

# Prisma 클라이언트 생성
npx prisma generate
```

### 환경 변수 설정

`.env.example` 파일을 참고하여 `.env` 파일을 생성하고 필요한 환경 변수를 설정하세요.

```bash
cp .env.example .env
```

필수 환경 변수:

- `DATABASE_URL`: PostgreSQL 연결 문자열
- `REDIS_URL`: Redis 연결 문자열
- `JWT_SECRET`: JWT 토큰 시크릿 키
- `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `AWS_S3_BUCKET_NAME`: AWS S3 설정
- OAuth 클라이언트 ID 및 시크릿 (Google, Kakao, Naver)

### 데이터베이스 설정

```bash
# 마이그레이션 실행
npm run migrate

# Prisma Studio 실행 (선택사항)
npm run studio
```

### 개발 서버 실행

```bash
# 개발 모드로 실행 (자동 재시작)
npm run dev
```

서버는 기본적으로 `http://localhost:4000`에서 실행됩니다.

### 프로덕션 빌드 및 실행

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 시작
npm start
```

### Docker를 사용한 실행

```bash
# Docker Compose로 실행
docker-compose up -d
```

### 기타 스크립트

```bash
# 린트 검사
npm run lint

# 타입 체크
npm run type-check

# 코드 포맷팅
npm run format

# 테스트 실행
npm test

# 테스트 감시 모드
npm run test:watch

# 시드 데이터 삽입 (개발용)
npm run seed
```
